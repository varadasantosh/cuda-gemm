GPU Memory Hierarchy & Processing Hierarchy

NVIDIA A100 Tensor Core GPU Architecture
Memory Hierarchy
REGISTERS - LEVEL 0
Capacity: 64K per SM
Latency: 1 cycle
Max per Thread: 255 registers
Scope: Per Thread
SHARED MEMORY - LEVEL 1
Capacity: 164KB per SM
Latency: ~20-30 cycles
Max per Block: 163KB
Scope: Per Thread Block
L1 CACHE - LEVEL 1
Combined: 192KB per SM
Latency: ~25-35 cycles
Type: Unified Cache
Scope: Per SM
L2 CACHE - LEVEL 2
Capacity: 40MB (A100)
Latency: ~200 cycles
Bandwidth: ~3TB/s
Scope: Entire GPU
HBM2 MEMORY - LEVEL 3 (Global Memory)
Capacity: 40/80GB
Latency: ~400-800 cycles
Bandwidth: 1.6-1.9TB/s
Processing Hierarchy
THREAD - EXECUTION UNIT
Definition: Single scalar processor
Resources: Up to 255 registers
Memory: Registers → Shared → Global
WARP - 32 THREADS
Definition: Group of 32 threads in SIMT
Scheduling: Basic unit on SM
Memory: Coalesced access patterns
Sync: __syncwarp() barrier
THREAD BLOCK - COOPERATIVE GROUP
Definition: Cooperating thread group
Size: 32 to 1024 threads
Max per SM: 32 blocks
Sync: __syncthreads() barrier
STREAMING MULTIPROCESSOR - PROCESSING CORE
CUDA Cores: 64 FP32, 64 INT32
Tensor Cores: 4
Max Warps: 64 concurrent
Schedulers: 4 warp schedulers
GPU DEVICE (A100) - COMPLETE PROCESSOR
Total SMs: 108 (of 128)
CUDA Cores: 6912 FP32/INT32 (108 * 64)
Tensor Cores: 432 (108 * 4)
NVLink 3.0: 600GB/s
