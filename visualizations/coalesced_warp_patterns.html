✓ Key Benefits of Coalesced Access
  • Consecutive memory access: All threads in warp access sequential addresses (0-31, 32-63, etc.)
  • Single cache line: 32 consecutive elements fit in 1-2 cache lines (128 bytes)
  • Efficient transactions: GPU combines all accesses into one or two memory transactions
